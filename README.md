## asianelevo.com website

First, run the development server:

```bash
#Initialise with 
npm install

#Developer Server
npm run dev

#Production build
#uncomment output: 'export' from next.config.js
npm run build
#out folder will contain Static Site Generated(SSG) for deployment.

```
